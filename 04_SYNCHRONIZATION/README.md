# Synchronization

This section demonstrates synchronization mechanisms in FreeRTOS using STM32F446RE.

## Projects

- Binary Semaphore
- Priority Inversion
- Stepper Motor using Binary Semaphore
- Mutex (Priority Inheritance)
- Counting Semaphore

**Hardware**
- STM32F446RE
- ARM Cortex-M4

**Concepts Covered**
- Binary Semaphore
- Mutex
- Priority Inheritance
- Counting Semaphore
