# FreeRTOS-STM32-Projects

Practical FreeRTOS implementations on the **STM32F446RE (ARM Cortex-M4)** covering task management, scheduling, synchronization, inter-task communication, software timers, RTOS drivers, CMSIS-RTOS v2, and scheduler behavior.

---

## Hardware Platform

- STM32F446RE Development Board
- ARM Cortex-M4
- ST-Link V2
- NEMA17 Stepper Motor
- DRV8825 Stepper Driver
- UART Terminal (CoolTerm)

---

## Development Environment

- STM32CubeIDE
- FreeRTOS
- CMSIS-RTOS v2
- Embedded C
